📌 AI-Based Candidate Screening System

This project is designed to help companies automatically screen resumes using Artificial Intelligence.

It takes resumes as input and compares them with job descriptions and on the basis of certain parameters the system calculates similarity scores and ranks candidates accordingly.

Working Process:

1. Load dataset

2. Clean resume text

3. Convert text into numerical format using TF-IDF

4. Apply machine learning model

5. Rank candidates

6. This system helps reduce manual effort and speeds up recruitment.
